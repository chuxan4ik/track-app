# Веб-приложение для трекинга доставок DeliveryTracker

Курсовая работа по дисциплине «Интернет-технологии»  
Направление подготовки 09.03.03 «Прикладная информатика»

---

## Описание проекта

Веб-приложение для отслеживания статуса доставок. Позволяет создавать записи о доставках, просматривать их список, изменять статус с автоматическим сохранением истории изменений, а также удалять записи. Реализован полный CRUD-цикл с веб-интерфейсом и REST API.

---

## Технологический стек

### Frontend:
- **HTML5** – структура страниц
- **CSS3** – адаптивная вёрстка (Flexbox, медиа-запросы)
- **JavaScript (ES6)** – взаимодействие с API, динамическое обновление интерфейса
- **Tera** – шаблонизатор для Rocket (аналог Jinja2)

### Backend:
- **Rust** – язык программирования (безопасность памяти, высокая производительность)
- **Rocket 0.5** – веб-фреймворк (асинхронный, безопасный)
- **SQLite** – встраиваемая реляционная база данных
- **sqlx** – асинхронный драйвер для работы с БД (проверка запросов на этапе компиляции)
- **Serde** – сериализация/десериализация JSON
- **Chrono** – работа с датой и временем
- **UUID** – генерация уникальных трек-номеров

---

## Реализованный функционал

-  **Создание доставки** – указание трек-номера (автогенерация при пустом поле), описания, отправителя, получателя
-  **Просмотр списка** – все доставки с отображением статуса, сортировка по дате создания
-  **Детальная информация** – просмотр полной информации о доставке и истории изменения статусов
-  **Изменение статуса** – выбор нового статуса из списка (Создана → Принята → В пути → Доставлена → Возврат) с автоматическим добавлением записи в историю
-  **Удаление доставки** – каскадное удаление вместе с историей
-  **Валидация** – проверка обязательных полей, уникальности трек-номера
-  **История изменений** – все статусы с временными метками
